06 / Limitations / Second-Best Alternative

Limitations / Second-Best Alternative

  • Sysimage failure: PackageCompiler cannot precompile ArchGDAL (upstream method overwriting bug). Mitigation: Remove ArchGDAL from sysimage packages list and test again, or use API-only solve path.
  • Pipeline materialize gap: Each case study's build script accepts different arguments. The pipeline currently passes --regime and --config uniformly, which fails for most case studies. A per-case-study mapping table is needed.
  • PowerShell compatibility: The pipelines were written for PS7+ but must run on PS5.1 (Windows default). Fixed, but testing is limited to the dry-run and --skip-solve paths.
  • API fallback path confirmed: All downstream modules (equity_irr, bess_dispatch) accept the normalized API v3 format.
