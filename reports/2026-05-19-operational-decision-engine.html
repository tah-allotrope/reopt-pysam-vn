06 / Limitations / Second-Best

  • 6 TOU scenarios remain unsolved. Financial outputs still show "no_results" until a solve completes.
  • Second-best alternative: NREL API as primary solver (avoids Julia dependency). Trade-off: rate limits, network reliance.
  • E2E test uses example results, not real solves. Baseline values are placeholders until true solve results exist.
  • Decree 146 demand charges use trial rates — Phase 3 rates not yet published (expected July 2026).
  • Capacity factor benchmark skips if PySAM not installed. Not all dev environments have it.
