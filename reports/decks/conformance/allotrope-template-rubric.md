## Rubric Dimensions & Scoring

Each dimension is scored 0–5:

| Score | Meaning |
|-------|---------|
| **5** | Exact match — indistinguishable from template |
| **4** | Close match — minor deviation, immediately fixable |
| **3** | Acceptable — conforms in spirit, but visible difference |
| **2** | Partial — some elements present, key aspects missing |
| **1** | Poor — attempted but largely non-conforming |
| **0** | Absent — not attempted or completely wrong |

**Scoring protocol:** Each deck is scored slide-by-slide against the dimension anchors below. The per-dimension score is the median across all applicable slides. The aggregate score is the sum across all 12 dimensions (max 60).

### D01 — Cover Slide Layout

**Template reference (Slide 1):**
- Logo (Allotrope) positioned top-left
- Title text centered: two lines max (`"Allotrope Vietnam Progress Update\nFor Tara"`-style)
- Subtitle/date line: `"February 2026"`
- Font: Cabin 26pt bold, color `#155B55`
- No bullet points or body text on cover

| Score | Anchor |
|-------|--------|
| 5 | Logo + centered title (2 lines max) + date line, Cabin 26pt bold `#155B55` |
| 4 | Logo + title + date present, font/color off by one attribute |
| 3 | All elements present but layout differs (e.g. left-aligned, extra text) |
| 2 | Logo missing or cover has bullet body text |
| 1 | Attempted a cover but no logo or wrong font entirely |
| 0 | No cover slide |

### D05 — Typography (Font Family & Sizes)

**Template reference:**
- **Font family:** Cabin throughout (headlines, body, footer, labels)
- **Headline size:** ~26pt (cover), ~23–24pt (section titles), ~19pt (slide titles)
- **Body size:** ~12pt (152400 EMU)
- **Card titles:** ~14pt bold
- **Footer:** ~8pt bold
- **Key numbers:** ~14–15pt bold

| Score | Anchor |

| 5 | Cabin used throughout, sizes match template roles within 1pt |
| 4 | Cabin used, sizes within 2pt of template roles |
| 3 | Different font family but sizes approximately correct, or Cabin used but sizes off |
| 2 | Wrong font family and sizes significantly off |
| 1 | Multiple font families mixed inconsistently |
| 0 | Font choices completely unrelated to template |
