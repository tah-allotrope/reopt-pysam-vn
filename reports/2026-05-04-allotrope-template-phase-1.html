06 / Limitations / Second-Best Alternative

Limitations / Second-Best Alternative

  • Template analysis is static. The python-pptx extraction reads XML structure but does not visually render slides. Subtle visual defects (misaligned elements, font rendering on non-standard systems) will only surface in Phase 4 review.
  • Manual download step. The template was downloaded via browser rather than automated API. This is acceptable for a one-time setup but would need automation for template versioning.
  • Second-best alternative: Using gws export feature directly (gws drive export) instead of browser manual download. The browser route was chosen because Google Slides export options are more reliably controlled via the UI.
